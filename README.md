# Boulder BioLabs Site

Production Next.js 16 site for Boulder BioLabs — contract medical device microbiology testing lab.

Part of the Boulder BioMed family of companies (https://boulderbiomed.com/).

## Stack

- Next.js 16 + App Router
- TypeScript
- Tailwind CSS v4
- No external runtime dependencies

## Run Locally

```
npm install
npm run dev
```

Open http://localhost:3000

## Compliance Language

NEVER write "ISO 11737-2 compliant", "ISO 17025 accredited", or "USP 71 certified". Use aligned-protocols language. The lab operates under the Boulder BioMed ISO 13485 QMS and does not yet hold separate lab-level accreditations.
